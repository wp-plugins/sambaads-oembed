== Description ==

Add sambaads to the available oEmbed providers

Adds the ability to paste a video URL from sambaads.com and have it automatically embed into a WordPress blog post, page or custom post type.

Simply paste the sambaads.com video URL into the content area and I will oEmbed into your post, page or custom post type.

Example:

	http://player.sambaads.com/iframe/00d24b737eb3be1f10f9af9094814c4d?m=6044

== Installation ==

1. Go to Dashboard >> Plugins >> Add New >> Search
2. Search for "sambaads oEmbed," click "Install Now," then "Activate Plugin"
3. There are no settings

== Frequently Asked Questions ==

= What is sambaads? =

Premium syndication content network. Look at <a href="http://sambaads.com.br">sambaads.com.br</a> and ask us to be an Producer or Publisher for our contents.

= Where are the settings? =

There are no settings. Activating sambaads oEmbeds enables oEmbed support for sambaads.com videos for all users.
